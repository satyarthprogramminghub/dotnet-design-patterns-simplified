# Series Overview

Welcome to **.NET Design Patterns Simplified**!

This video series teaches the most useful design patterns using .NET 8 with practical but simple examples.

### Why Learn Design Patterns?
- Write more maintainable and testable code
- Solve common problems in a proven way
- Better understand ASP.NET Core and DI

### Learning Approach
- One pattern per video
- Show "Before" (problematic code) → "After" (with pattern)
- Use the same **Order Management** domain across examples
- Keep examples short, clear, and focused on the core concept
- Heavy use of .NET built-in features (especially DI)

### Tech Stack
- .NET 10 (.NET 8 or .NET 9 are also fine)
- C# 12+
- Minimal console demos + occasional Minimal API examples
- Built-in Dependency Injection
